# OpoCoach

OpoCoach es una plataforma de preparación de oposiciones basada en el temario oficial de una convocatoria.

La documentación del proyecto se organiza de la siguiente forma:

| Documento | Contenido |
|-----------|-----------|
| `VISION.md` | Objetivo, principios y visión general del proyecto. |
| `FUNCIONALIDAD.md` | Descripción funcional de la aplicación. |
| `ARQUITECTURA.md` | Arquitectura general del sistema y de la base de datos. |
| `ESTADO.md` | Estado actual del desarrollo. |
| `CONVENCIONES.md` | Normas de desarrollo y estilo. |
| `ESTRUCTURA_PROYECTO.md` | Organización de carpetas y módulos. |
| `MANTENIMIENTO.md` | Procesos de mantenimiento y auditoría. |
| `SCRIPTS.md` | Inventario de scripts del proyecto. |

La documentación describe el estado actual del proyecto y deberá mantenerse sincronizada con su evolución.
