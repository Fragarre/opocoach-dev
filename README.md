# OpoCoach

OpoCoach es una plataforma para la preparación de oposiciones basada en el temario oficial de una convocatoria.

El proyecto permite importar legislación, exámenes oficiales y bancos de preguntas, clasificarlos, validarlos y generar simulacros de examen apoyados por IA.

## Características

- Importación de legislación y documentación oficial.
- Importación de exámenes oficiales.
- Clasificación automática de preguntas.
- Validación mediante reglas e IA.
- Generación de bancos de preguntas.
- Generación de simulacros en PDF.
- Aplicación web basada en Streamlit.
